# 제품 요구사항 문서 (PRD): AI 기반 자동 릴스 생성기 (Auto Reels Creator)

---

### **1. 개요 (Introduction)**

*   **1.1. 문제 정의 (Problem Statement)**
    인스타그램 릴스, 틱톡, 유튜브 쇼츠와 같은 숏폼 콘텐츠는 마케팅에서 매우 중요하지만, 고품질의 영상을 꾸준히 제작하는 것은 많은 시간과 창의적인 노력이 필요하다. 특히 주제 선정, 대본 작성, 미디어 수집, 나레이션 생성, 영상 편집 과정을 매번 수동으로 진행하는 것은 매우 비효율적이다.

*   **1.2. 해결 방안 (Proposed Solution)**
    사용자가 입력한 주제를 기반으로 AI(Gemini, Groq)가 대본을 작성하고, 관련성 높은 스톡 영상과 배경음악을 자동으로 수집하며, TTS 나레이션을 입혀 최종적인 9:16 비율의 릴스 영상을 자동으로 조립하는 통합 솔루션을 제공한다.

*   **1.3. 제품 비전 (Vision)**
    누구나 아이디어만 있으면 전문적인 영상 편집 기술 없이도 고품질의 숏폼 콘텐츠를 즉시 생성할 수 있는 AI 영상 제작 자동화 도구.

### **2. 대상 사용자 (Target User)**

*   **주요 사용자:** 숏폼 콘텐츠 크리에이터, 소셜 미디어 마케터, 개인 브랜드 운영자
*   **2차 사용자:** 영상 편집 시간을 단축하고자 하는 전문 편집자, 스타트업 마케팅팀

### **3. 제품 요구사항 및 기능 (Product Requirements & Features)**

#### **F-1: AI 기반 대본 생성 (AI Scripting)**
*   **FR-1.1:** 주제와 목표 길이를 입력받아 AI(Gemini 1.5/2.0 또는 Groq Llama 3)를 통해 숏폼에 최적화된 대본을 생성한다.
*   **FR-1.2:** 대본은 Hook(시선 강탈), Value(정보 제공), CTA(참여 유도) 구조를 따르며, 장면별 시각적 설명(Visual Description)과 키워드를 포함한다.

#### **F-2: 미디어 자동 수집 및 검색 (Media Sourcing)**
*   **FR-2.1:** Pexels API를 사용하여 대본의 장면 키워드와 일치하는 고화질 9:16 영상을 자동으로 검색하고 다운로드한다.
*   **FR-2.2:** AI가 다운로드된 미디어의 메타데이터를 분석하여 대본 맥락과 일치하는지 검증하고, 부적절할 경우 재검색을 시도한다.

#### **F-3: 오디오 생성 및 합성 (Audio Processing)**
*   **FR-3.1:** Edge-TTS를 사용하여 한국어 나레이션을 생성하고, 나레이션 길이에 맞춰 장면의 재생 시간을 자동으로 조절한다.
*   **FR-3.2:** 대본 분위기에 맞는 배경음악(BGM)을 Pixabay 등에서 수집하거나 기존 에셋에서 선별하여 합성한다.
*   **FR-3.3:** 장면 전환 시 SFX(웅 소리 등)를 자동으로 삽입하여 역동성을 높인다.

#### **F-4: 자동 영상 조립 및 렌더링 (Video Assembly)**
*   **FR-4.1:** MoviePy를 사용하여 각 장면, 나레이션, BGM, 자막을 9:16 비율(1080x1920)로 조립한다.
*   **FR-4.2:** 정지 영상(이미지) 사용 시 Ken Burns 효과(줌/팬)를 적용하여 정적인 느낌을 제거한다.
*   **FR-4.3:** 텍스트 오버레이(자막)를 중앙 하단에 가독성 높게 배치한다.

#### **F-5: 사용자 인터페이스 (GUI)**
*   **FR-5.1:** Tkinter 기반의 데스크톱 앱을 제공하여 비전공자도 쉽게 주제 입력 및 생성 과정을 관리할 수 있도록 한다.

### **4. 기술 사양 (Technical Specifications)**

*   **언어:** Python 3.9+
*   **AI 모델:** Google Gemini 1.5 Flash/Pro, Groq Llama 3.3 70B
*   **핵심 라이브러리:** `moviepy`, `edge-tts`, `google-generativeai`, `groq`, `requests`, `pillow`
*   **외부 API:** Pexels API

### **5. 성공 지표 (Success Metrics)**

*   **생성 성공률:** 입력부터 렌더링 완료까지 오류 없이 도달하는 비율 (목표: 95% 이상)
*   **맥락 일치도:** AI 검증을 통한 장면과 자막의 일관성 유지.
*   **제작 속도:** 30초 내외의 영상을 5분 이내에 완성 (네트워크/하드웨어 성능에 따라 변동).
